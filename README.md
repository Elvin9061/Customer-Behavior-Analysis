# Customer-Behavior-Analysis

# Overview

This project focuses on analyzing customer shopping behavior using Python, MySQL, and Power BI. The objective of the project is to clean, explore, and analyze customer purchase data to identify trends, customer patterns, and business insights that can support better decision-making.

The project includes:
- Data loading and preprocessing using Python
- Exploratory Data Analysis (EDA)
- Data cleaning and transformation
- SQL analysis using MySQL Workbench
- Interactive dashboard creation in Power BI
- Final report generation with key findings

# Tools & Technologies Used

| Tool | Purpose |
|---|---|
| Python | Data cleaning and EDA |
| Pandas & NumPy | Data manipulation |
| Matplotlib & Seaborn | Data visualization |
| MySQL Workbench | SQL analysis |
| Power BI | Dashboard creation |

# Project Steps
1. Data Loading
 - Imported dataset into Python using Pandas
 - Checked dataset structure, columns, and data types

2. Data Cleaning
 - Handled missing values
 - Removed duplicates
 - Corrected inconsistent data
 - Converted data types where necessary

3. Exploratory Data Analysis (EDA)

  Performed analysis to identify:

 - Customer purchasing trends
 - Most popular product categories
 - Spending behavior
 - Payment method preferences
 - Relationship between variables

4. SQL Analysis (MySQL Workbench)
 - Imported cleaned data into MySQL
 - Wrote SQL queries for business analysis
 - Used filtering, grouping, joins, and aggregate functions
 - Generated insights from customer purchase behavior

5. Power BI Dashboard

  Created an interactive dashboard to visualize:

 - Total sales
 - Customer distribution
 - Product category performance
 - Purchase trends
 - Customer behavior insights

# Results & Insights

Some key insights identified from the analysis:

- Most purchased product categories
- Customer spending trends
- Popular payment methods
- High-value customer segments
- Shopping behavior patterns across different demographics

These insights can help businesses improve:

- Marketing strategies
- Customer targeting
- Product recommendations
- Sales performance
